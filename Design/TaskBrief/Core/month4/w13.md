**Git branch:** `feature/W13-ui-foundation`

> **Cross-cutting constraints.** All types conform to `Design/Modules/Other/CanonicalTypes.md` (single source — never re-declare a shape). Pin runtime deps to **exact** versions (no `^`/`~`). `tsconfig` `strict: true`, no `any` (use `unknown` + `zod.parse()` at I/O). Route `/` stays the workspace editor. ≤200 lines/file (`VibeCode.md` §3); inter-module imports only via each module's `index.ts` public surface; CSS uses only `var(--rs-*)` tokens (`DesignSystem_Tokens.md`).
>
> **🎨 Frontend discipline (`Design/Frontend/`).** Mỗi component dựng **đúng spec** `Frontend/2.Components/<X>.md` theo khuôn `_ComponentSpecRule.md` (Anatomy · Variants · **States đủ** · A11y · Motion · Do/Don't). Đối chiếu `0.ArtDirection.md` §11 checklist trước khi coi là xong. Icon = Lucide (`Frontend/Other/Icons.md`); microcopy = `Frontend/Other/VoiceAndContent.md`; motion ≤200ms + `prefers-reduced-motion` (`Frontend/Other/Motion.md`).
>
> **W13 install matrix.** Adds **runtime** UI deps (exact pin, approve trước khi cài): `@radix-ui/react-dialog`, `@radix-ui/react-tabs`, `@radix-ui/react-toast`, `@radix-ui/react-select`, `lucide-react` (`TechnicalStack §2b/§8c`). Chạy `npm install --save-exact` để đồng bộ `package-lock.json`. **FORBIDDEN:** UI kit có style (MUI/Ant/Chakra/Mantine), Tailwind, CSS-in-JS runtime, animation lib (Framer/GSAP) — `TechnicalStack §2b` CẤM.
>
> **🔒 Trình bày, không đổi behavior/shape.** W13 chỉ **thêm** tầng UI primitive `src/components/ui/`; **không** đổi logic module, **không** đổi `CanonicalTypes`, **không** đụng pipeline/checker/export. Panel cũ chưa refactor tuần này (đó là W14).
>
> **🔒 Headless = ta style.** Radix dùng **không** style mặc định — mọi màu/spacing/radius/elevation lấy từ token. Không hardcode hex/px. Giữ nguyên a11y Radix lo sẵn (focus-trap/keyboard/ARIA), chỉ bổ sung `aria-label` nội dung.

> **⚠️ Reconciliation — token đã có, không sinh mới.** Component đọc `var(--rs-*)` sẵn có (`DesignSystem_Tokens.md` §2–§6). Nếu thiếu token (vd `--rs-motion-*`), **bổ sung vào file token trước** rồi mới dùng (Single Source of Truth), không hardcode.
>
> **⚠️ Reconciliation — không refactor panel tuần này.** W13 dựng primitive + gallery; việc thay primitive vào Write/Format/Check/Export/Present panel là **W14**. Tránh đổi behavior khi đang xây nền.
>
> **⚠️ Reconciliation — Select/Textarea tách khỏi Input.** Theo quyết định Frontend (B4): `Input` chỉ text/number; `Select` (Radix) và `Textarea` là component riêng.

## W13 Locked Decisions (review-agreed)

1. **Headless + token-only.** Radix lo a11y/behavior; style 100% bằng `var(--rs-*)`; không hardcode hex/px; không UI kit có style (`TechnicalStack §2b`).
2. **States là điều kiện done.** Mỗi component cover đủ trạng thái theo `_ComponentSpecRule.md` §3 (loading/empty/error/disabled/focus); thiếu state = chưa xong.
3. **Không refactor panel.** W13 chỉ dựng primitive + gallery; thay vào module panel là W14; không đổi behavior/shape/CanonicalTypes.
4. **Select/Textarea riêng (B4).** Input chỉ text/number.
5. **Icon Lucide, microcopy chuẩn.** Icon từ `Frontend/Other/Icons.md`; nhãn/copy từ `VoiceAndContent.md`; Việt 100% (giữ Markdown/PDF/DOCX/HTML/TOC).
6. **Deps runtime exact pin.** Radix ×4 + lucide-react exact; `npm install --save-exact`; commit lockfile.
