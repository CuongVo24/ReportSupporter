## W9 Locked Decisions (review-agreed)

1. **Present đọc canonical, không đổi shape.** Đọc `ReportProject`/`ReportSection`/`EvidenceItem`; type Present mới thêm vào CanonicalTypes §9 trước khi code (`5.Present.md §3`).

2. **Deterministic, offline, no AI.** Outline/timeline/speaker baseline sinh không cần mạng/AI; AI để W11 sau explicit action (RoadMap W11, `ProductPRD §6`).

3. **Outline giữ đúng dòng chương mục.** Duyệt `ReportSection[]` theo `order` + heading đánh số (Module 2); section rỗng bỏ, bullets cap để slide không quá tải (`5.Present.md §6`).

4. **Speaker từ `metadata.members`, edge-safe.** Normalize string|string[]; thiếu members → 1 speaker mặc định / để trống; chia slide deterministic.

5. **Timeline một nguồn từ outline.** `totalSeconds` = Σ slots; cảnh báo khi vượt limit; immutable, không mutate outline.

6. **Không cài `pptxgenjs`/AI tuần này.** Export PPTX/PDF slides + AI rewrite là phase sau (cần Contract/approve riêng); W9 chỉ logic + panel.

7. **Public surface `@/modules/present`.** Mọi import liên-module qua `present/index.ts`; không reach-in file nội bộ.
