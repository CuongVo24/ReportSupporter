# Task Brief Week 11 - Core Team

**Module target:** `src/modules/write/ai/` + `src/modules/present/ai/` — **Optional AI Assistant**: lớp AI tùy chọn (feature flag mặc định OFF) phủ lên Module 1 (Write) + Module 5 (Present): AI-assisted outline, section rewrite suggestion, academic tone improvement. AI **chỉ chạy sau explicit user action**, **giữ user content control luôn hiển thị**, app W1–W10 **chạy đủ tính năng khi tắt AI** (RoadMap Phase 3 / W11).
**Git branch:** `feature/W11-ai-assistant`

> **Cross-cutting constraints.** All types conform to `Design/Modules/Other/CanonicalTypes.md` (single source — never re-declare a shape). Pin runtime deps to **exact** versions (no `^`/`~`). `tsconfig` `strict: true`, no `any` (use `unknown` + `zod.parse()` at I/O). Route `/` stays the workspace editor. ≤200 lines/file (`VibeCode.md` §3); inter-module imports only via each module's `index.ts` public surface; CSS uses only `var(--rs-*)` tokens.
>
> **W11 install matrix.** **Mặc định KHÔNG cài.** AI provider/SDK **nằm ngoài** `TechnicalStack.md` → chọn provider/SDK là **lib/integration ngoài stack đã khoá**, **bắt buộc xin approve trước khi cài** (`VibeCode.md §4` cấm import lậu; `TechnicalStack` AI RULE). Tuần này thiết kế lớp AI **provider-agnostic** sau cổng explicit-action; integration cụ thể chỉ làm sau khi user duyệt provider + cách lưu key (không hard-code secret). **FORBIDDEN:** hard-code API key/secret; AI client cài "cho tiện" trước khi approve.
>
> **🔒 AI sau explicit user action (`ProductPRD.md §6`, MasterRoadMap W11).** Mọi AI action chỉ chạy khi user bấm nút rõ ràng — **không** auto-run nền, **không** auto-rewrite. Suggestion hiển thị dạng đề xuất accept/reject, **không tự ghi đè** nội dung.
>
> **🔒 AI là optional, không phải lõi.** Feature flag mặc định **OFF**; tắt AI → toàn bộ app W1–W10 chạy đủ tính năng (off-state test). AI là điểm cộng, không phải điều kiện dùng sản phẩm.
>
> **🔒 User content control luôn hiển thị.** Luôn thấy bản gốc + diff gốc↔đề xuất + nút hoàn tác (`UserControlBar`). User giữ quyền kiểm soát nội dung.

> **⚠️ Reconciliation — chỉ 3 tính năng AI.** Đúng outline-assist / rewrite-section / improve-tone từ MasterRoadMap W11; không thêm tính năng AI ngoài 3 mục này.
>
> **⚠️ Reconciliation — provider-agnostic gateway.** `ai-gateway.ts` là interface trừu tượng (không hard-code provider). Integration provider thật chỉ sau approve; tuần này nếu chưa approve → gateway trả trạng thái "chưa cấu hình" và UI vẫn vận hành (nút disabled + ghi chú).
>
> **⚠️ Reconciliation — đọc artifact W9/W10, không re-generate.** Outline-assist nâng cấp `SlideOutline[]` (W9); rewrite/tone đọc `ReportSection` (W2 editor); không viết lại generator deterministic.

## W11 Locked Decisions (review-agreed)

1. **AI sau explicit user action.** Mọi AI action sau click rõ ràng; không auto-run/auto-rewrite; suggestion accept/reject, không tự ghi đè (`ProductPRD.md §6`, MasterRoadMap W11).
2. **AI optional, flag OFF mặc định.** Tắt AI → app W1–W10 đủ tính năng (off-state test); AI là điểm cộng.
3. **Provider-agnostic, không cài lậu.** `ai-gateway.ts` trừu tượng; chọn provider/SDK = ngoài stack → Contract + approve trước khi cài; không hard-code key (`TechnicalStack` AI RULE).
4. **User content control luôn hiển thị.** Bản gốc + diff + undo luôn thấy (`UserControlBar`).
5. **Chỉ 3 tính năng AI.** outline-assist / rewrite-section / improve-tone; không thêm.
6. **Đọc artifact W9/W10/W2, không re-generate.** Nâng cấp outline/section có sẵn; không viết lại generator deterministic.

## Risks

| Risk | Level | Mitigation |
|---|---:|---|
| AI tự chạy / tự ghi đè nội dung | High | Mọi AI action sau explicit click; suggestion accept/reject, không auto-write (Locked #1). |
| AI trở thành bắt buộc | High | Flag OFF mặc định; off-state test app vẫn đủ tính năng (Locked #2). |
| Cài SDK ngoài stack không xin phép | High | Provider/SDK = ngoài stack → Contract + approve; gateway provider-agnostic (Locked #3). |
| Lộ/secret key hard-code | High | Không hard-code key; user tự cấu hình; gateway không nhúng secret. |
| Mất user content control | Medium | `UserControlBar` luôn hiện bản gốc + undo + diff (Locked #4). |
| Re-declare type AI (drift) | Low | Thêm `CanonicalTypes §10` trước; import `@/types`. |
