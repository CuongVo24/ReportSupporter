# Module 5 - Present

## Purpose

Present turns a finished report into presentation support material.

This module is not part of the first MVP implementation, but it shapes the product direction.

## Later Responsibilities

- Generate slide outline from report sections.
- Generate speaker script.
- Generate defense Q&A.
- Assign speakers by section.
- Export PPTX and PDF slides.
- Create final submission checklist.

## Acceptance Criteria For Later Phase

- Slide outline keeps the same chapter flow as the report.
- Generated script references key screenshots and evidence.
- Q&A focuses on likely defense questions for project reports.
