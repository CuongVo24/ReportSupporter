# Module 2 - Format

## Purpose

Format turns raw Markdown into a report that looks like it was manually aligned in a word processor.

## MVP Responsibilities

- Normalize heading numbering.
- Generate table of contents data.
- Apply A4 layout presets.
- Apply academic typography presets.
- Normalize figure and table captions.
- Prepare content for PDF and DOCX exporters.

## Formatting Presets

- Page: A4.
- Margin: configurable, default academic preset.
- Font: Times New Roman or configurable equivalent.
- Font size: 13 or 14.
- Line height: 1.5.
- Text alignment: justified for body content.
- Chapter behavior: start new major chapter on a new page in PDF/DOCX output.

## Acceptance Criteria

- Exported HTML preview uses the same heading/caption numbering as PDF/DOCX.
- Figures and tables can be referenced by generated numbers.
- Formatting logic is deterministic from Markdown + metadata.
