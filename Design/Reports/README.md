# Reports

This folder stores delivery evidence for ReportSupporter.

Recommended structure:

```text
Reports/
  Month1/
    W1/
      W1_QA_Report.md
      build_output.txt
      screenshots/
  Month2/
  Month3/
```

Use this folder for:

- QA summaries.
- Build logs.
- Export samples.
- Screenshot evidence.
- Checker result samples.
- Acceptance reports.
